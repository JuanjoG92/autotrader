¿Cómo conectar tu exchange?
  1. Ingresá a la sección de API de tu exchange (Bybit, Binance, KuCoin)
  2. Creá una nueva API Key (tipo "Generada por el sistema")
  3. Habilitá "Habilitar lectura" + "Habilitar trading de márgenes y spot"
  4. En "Restricciones de IP" agregá: 172.96.8.245
  5. Copiá la API Key y el Secret y pegalos acá abajo
🔒 Seguridad: La restricción por IP asegura que solo nuestro servidor pueda usar tu API Key.
